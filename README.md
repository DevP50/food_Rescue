 NoWasteAm 🍽️
A food rescue platform that connects people to surplus food from restaurants and legally capable individuals.

## Features
- Add food listings
- Request food
- Reserve system
- Confirmation System (By Calling the Restaurant to confirm pickup details )
- Status tracking (Available, Urgent, Reserved)
-Payment Services

## Tech Stack
- Flask (Python)
- HTML/CSS
- JavaScript
